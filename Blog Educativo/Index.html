🏗️ Etiquetas Estructurales

<header>

Definición: Representa el encabezado de una página o sección.

Sintaxis:

<header>Contenido</header>

Ejemplo:

Título del sitio

Buenas prácticas: Puede haber varios <header>, pero usarse de forma semántica.

<nav>

Definición: Sección que contiene enlaces de navegación.

Sintaxis:

<nav>Enlaces</nav>

Ejemplo:

Inicio | Servicios

Buenas prácticas: Usarlo solo para enlaces importantes de navegación.

<main>

Definición: Contiene el contenido principal del documento.

Sintaxis:

<main>Contenido principal</main>

Ejemplo:

Contenido principal

Buenas prácticas: Solo debe existir un <main> por documento.

<section>

Definición: Define una sección temática dentro del documento.

Sintaxis:

<section>Contenido</section>

Ejemplo:

Sección de ejemplo

Contenido relacionado.

Buenas prácticas: Debe incluir un encabezado (<h2>, <h3>, etc.).

<article>

Definición: Representa contenido independiente y autónomo.

Sintaxis:

<article>Contenido independiente</article>

Ejemplo:

Título del artículo

Contenido del artículo.

Buenas prácticas: Puede contener su propio <header> y <footer>.

<aside>

Definición: Contiene información complementaria al contenido principal.

Sintaxis:

<aside>Contenido secundario</aside>

Ejemplo:

Información adicional.

Buenas prácticas: Debe complementar el contenido principal.

<footer>

Definición: Representa el pie de página de una sección o documento.

Sintaxis:

<footer>Información final</footer>

Ejemplo:

© 2026 Mi Sitio Web

Buenas prácticas: Puede haber varios <footer> dentro de un documento.

✏️ Etiquetas de Texto

<p>

Definición: Define un párrafo de texto.

Sintaxis:

<p>Texto</p>

Ejemplo:

Este es un párrafo de ejemplo.

<h1> a <h6>

Definición: Etiquetas de encabezado. <h1> es el más importante y <h6> el menor.

Sintaxis:

<h1>Título principal</h1>
                <h2>Subtítulo</h2>

Ejemplo:

Título H1

Título H2

Título H3

Buenas prácticas: Solo un <h1> por página y respetar el orden jerárquico.

<span>

Definición: Etiqueta en línea para aplicar estilos a una parte del texto.

Sintaxis:

<span>Texto</span>

Ejemplo:

Este texto tiene una palabra resaltada.

<strong>

Definición: Indica que el texto tiene importancia fuerte (negrita semántica).

Sintaxis:

<strong>Texto importante</strong>

Ejemplo:

Este texto es muy importante.

<em>

Definición: Indica énfasis en el texto (cursiva semántica).

Sintaxis:

<em>Texto enfatizado</em>

Ejemplo:

Este texto tiene énfasis.

<hr>

Definición: Representa un cambio temático entre párrafos (línea horizontal).

Sintaxis:

<hr>

Ejemplo:

Buenas prácticas: Es autocontenida, no necesita cierre.

📋 Etiquetas de Listas

<ul> y <li>

Definición: Crean una lista desordenada (con viñetas).

Sintaxis:

<ul>
                <li>Elemento 1</li>
                <li>Elemento 2</li>
                </ul>

Ejemplo:

  • Elemento 1
  • Elemento 2
📊 Etiqueta Tabla

<td>

Definición: Define una celda de datos dentro de una fila (<tr>) en una tabla.

Sintaxis:

<td>Dato</td>

Ejemplo:

Mariana 18

Buenas prácticas: Siempre debe estar dentro de un <tr>.

🎬 Etiquetas Multimedia

<img>

Definición: Inserta una imagen en la página.

Sintaxis:

<img src="imagen.jpg" alt="Descripción">

Ejemplo:

Buenas prácticas: Siempre incluir el atributo alt para accesibilidad.

<iframe>

Definición: Incrusta otro documento o video dentro de la página.

Sintaxis:

<iframe src="URL" width="300" height="200"></iframe>

Ejemplo:

Buenas prácticas: Incluir siempre el atributo title.

<video>

Definición: Inserta un video directamente en la página.

Sintaxis:

<video src="video.mp4" controls></video>

Ejemplo:

Buenas prácticas: Usar controls para que el usuario pueda reproducirlo.

<audio>

Definición: Inserta un archivo de audio en la página.

Sintaxis:

<audio src="audio.mp3" controls></audio>

Ejemplo:

Buenas prácticas: Usar <source> para ofrecer varios formatos (mp3, ogg).
