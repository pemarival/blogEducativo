Etiquetas Estructurales

<header>

Definición: Representa el encabezado de una página o sección. Contiene información introductoria.

Sintaxis:

<header>Contenido</header>

Ejemplo:

Título del sitio

Uso: Se utiliza para mostrar títulos, logotipos o navegación.

Buenas prácticas: Puede haber varios <header>, pero deben usarse de forma semántica.

<nav>

Definición: Representa una sección que contiene enlaces de navegación.

Sintaxis:

<nav>Enlaces</nav>

Ejemplo:

Inicio | Servicios

Uso: Se utiliza para menús principales o secundarios.

Buenas prácticas: Usarlo solo para enlaces importantes de navegación.

<main>

Definición: Contiene el contenido principal del documento.

Sintaxis:

<main>Contenido principal</main>

Ejemplo:

Contenido principal

Uso: Encierra la información más importante de la página.

Buenas prácticas: Solo debe existir un <main> por documento.

<section>

Definición: Define una sección temática dentro del documento.

Sintaxis:

<section>Contenido</section>

Ejemplo:

Sección de ejemplo

Contenido relacionado.

Uso: Agrupa contenido relacionado bajo un mismo tema.

Buenas prácticas: Debe incluir un encabezado (<h2>, <h3>, etc.).

<article>

Definición: Representa contenido independiente y autónomo.

Sintaxis:

<article>Contenido independiente</article>

Ejemplo:

Título del artículo

Contenido del artículo.

Uso: Se utiliza para noticias, publicaciones o entradas de blog.

Buenas prácticas: Puede contener su propio <header> y <footer>.

<aside>

Definición: Contiene información complementaria relacionada con el contenido principal.

Sintaxis:

<aside>Contenido secundario</aside>

Ejemplo:

Información adicional.

Uso: Se utiliza para contenido lateral como anuncios o enlaces relacionados.

Buenas prácticas: Debe complementar el contenido principal.

<footer>

Definición: Representa el pie de página de una sección o documento.

Sintaxis:

<footer>Información final</footer>

Ejemplo:

© 2026 Mi Sitio Web

Uso: Se coloca al final de una página o sección.

Buenas prácticas: Puede haber varios <footer> dentro de un documento.

Etiquetas de Texto

<p>

Definición: Define un párrafo.

<p>Texto</p>

Ejemplo práctico:

Este es un párrafo de ejemplo.

<h1> a <h6>

Definición: Son etiquetas de encabezado que definen títulos y subtítulos. <h1> es el más importante y <h6> el de menor jerarquía.

Sintaxis:

        <h1>Título principal</h1>
        <h2>Subtítulo</h2>

Ejemplo:

Título H1

Título H2

Título H3

Título H4

Título H5
Título H6

Uso: Organizar el contenido por niveles de importancia.

Buenas prácticas: Solo debe existir un <h1> principal por página y respetar el orden jerárquico.

<span>

Definición: Etiqueta en línea que se utiliza para aplicar estilos o modificar una parte del texto.

Sintaxis:

<span>Texto</span>

Ejemplo:

Este texto tiene una palabra resaltada.

Uso: Aplicar estilos CSS o manipulación con JavaScript a partes pequeñas del texto.

Buenas prácticas: No tiene significado semántico, usar solo cuando sea necesario para estilo o script.

<strong>

Definición: Indica que el texto tiene importancia fuerte.

Sintaxis:

<strong>Texto importante</strong>

Ejemplo:

Este texto es muy importante.

Uso: Resaltar contenido importante.

Buenas prácticas: Usarlo por significado semántico, no solo para poner negrita.

<em>

Definición: Indica énfasis en el texto.

Sintaxis:

<em>Texto enfatizado</em>

Ejemplo:

Este texto tiene énfasis.

Uso: Destacar palabras o frases con énfasis.

Buenas prácticas: Usarlo por significado semántico, no solo para cursiva.

<hr>

Definición: Representa un cambio temático entre párrafos.

Sintaxis:

<hr>

Ejemplo:

Uso: Separar contenido visual o temáticamente.

Buenas prácticas: Es una etiqueta autocontenida, no necesita cierre (</hr>).

Etiqueta Tabla

<td>

Definición: Define una celda de datos dentro de una fila (<tr>) en una tabla.

Sintaxis:

    <td>Dato</td>

Ejemplo:

Mariana 18

Uso: Contener información o datos dentro de una tabla.

Buenas prácticas:Debe estar siempre dentro de una etiqueta <tr>.

Etiquetas Multimedia

<img>

Definición: Permite insertar una imagen dentro de una página web.

Sintaxis:

<img src="ruta-imagen.jpg" alt="Descripción de la imagen">

Ejemplo:

Uso: Mostrar imágenes como fotografías, ilustraciones, logotipos o gráficos.

Buenas prácticas:Siempre incluir el atributo alt para accesibilidad.

<iframe>

Definición: Permite incrustar otro documento HTML dentro de la página actual, como videos, mapas o páginas externas.

Sintaxis:

<iframe src="URL" width="ancho" height="alto"></iframe>

Ejemplo:

Uso: Insertar contenido externo como videos de YouTube, mapas de Google Maps u otras páginas web.

Buenas prácticas: - Siempre incluir el atributo title para accesibilidad. - Definir ancho y alto.

<video>

Definición: Permite insertar contenido de video directamente en una página web sin necesidad de complementos externos.

Sintaxis:

<video src="video.mp4" controls>
</video>

Ejemplo:

Uso: Mostrar videos alojados en el servidor o en rutas locales del proyecto.

Buenas prácticas: Incluir el atributo controls para permitir reproducción, pausa y volumen y tambien definir width y height.

<audio>

Definición: Permite insertar contenido de audio en una página web sin necesidad de complementos externos.

Sintaxis:

<audio src="audio.mp3" controls>
</audio>

Ejemplo:

Uso: Reproducir archivos de sonido como música, podcasts o efectos de audio.

Buenas prácticas: - Incluir el atributo controls para mostrar los controles de reproducción. - Utilizar la etiqueta <source> para ofrecer varios formatos (mp3, ogg).
